# Find-Your-Shell
- Find-Your-Shell是一款由go语言开发得网站webshell查杀工具,支持linux,windows,默认包含了一部分php规则,
- 用户可以自定义添加规则对指定类型得文件进行查杀。
***
## 安装与使用
- `find-your-shell.exe 或 findyourshell `启动文件 
- `conf.db` 规则与配置文件，复制可以跨电脑使用
